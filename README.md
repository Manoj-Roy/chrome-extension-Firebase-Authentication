# chrome-extension-Firebase-Authentication

Need to add Firebase configuration on `firebase.js` file

```bash
const firebaseConfig = {
    apiKey: "", 
    authDomain: "",
    projectId: "", 
    storageBucket: "",
    messagingSenderId: "",
    appId: "", 
    measurementId: ""
};
```

Open Chrome 
```bash
Extension Manage => Load Unpacked => choose Extension folder 
```
Authentication features Done

```bash
Sign Up, 
Sign In, 
Logout
```
